# 次に確認したい質問事項

最終更新: 2026-03-09

## 0. この章の目的

- ここまでの世界観構築を踏まえて、今後ユーザーに確認したい論点をまとめる
- 優先度の高いものから順に確認できるようにする
- 好みの問題と仕様固定の問題を分けて整理する

## 1. 優先度高

### 1.1 教会の最終温度感

- 確定:
- 「公共性もあるが、体面や権威も重い」中庸で固定

### 1.2 遺跡の最終トーン

- 確定:
- 神秘と実用品の中間で固定し、完全SFには寄せない

### 1.3 他ギルドとの関係の強度

- 確定:
- 競争中心で、一部に敵対や警戒が混じる前提で固定

## 2. 優先度中

### 2.1 教会・商会・市参事会の三すくみの濃さ

- 確定:
- 三者の力関係がある前提で維持する

### 2.2 異常系危険生物の濃さ

- 確定:
- 現状の濃さで維持し、異常系を過度に増やさない

### 2.3 依頼主の「嫌な感じ」の強さ

- 確定:
- あまり嫌な人間を出しすぎず、事情優先で描く

### 2.4 冒険者の生活苦の重さ

- 確定:
- 生活事情は残しつつ、全体は暗くしすぎない

## 3. 優先度低

### 3.1 都市外のさらに広い世界

- 確定:
- 当面はエンベルク都市圏まわりに集中し、広域世界は背景止まりでよい

### 3.2 祭事の具体数

- 確定:
- 大きな祭事は年1回の `灯河祭` を基準とする

### 3.3 プレイヤーギルド拠点の日常描写量

- 確定:
- 依頼と面談中心で、生活描写は背景寄りに維持する

## 4. 現時点で未確定の論点

- 他ギルド別の運営方針の細かさ
- 依頼主の満足・不満・再依頼文のトーン幅
- 冒険者自己紹介テンプレートの粒度
- 中盤案件プールの広げ方
- 地域顔役テンプレートの詳細化レベル
