# 退避 旧世界観案 案内

最終更新: 2026-03-10

このディレクトリは、以前の中世ファンタジー寄り世界観を前提に作っていた文書群の退避置き場です。
現在の正本ではありませんが、設定や語彙の参照用として残しています。

## 現在の扱い

- 削除対象ではない
- ただし、いま優先して更新する場所ではない
- 現行仕様は `Docs/現行_世界観差し替え設計/` を参照する
