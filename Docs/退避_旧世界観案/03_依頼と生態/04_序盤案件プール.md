# 序盤案件プール

最終更新: 2026-03-09

## 0. この章の目的

- 新設ギルドの序盤で自然に受けうる案件群を具体化する
- 世界観、地方帯、依頼主カテゴリ、季節変動と整合する案件のたたき台を持つ
- AI生成や手動調整の両方に使える「序盤案件の骨格」を用意する

## 1. 序盤案件の設計方針

- 新設ギルドでも受けられる規模であること
- 既存ギルドが取りこぼしうる理由があること
- 地域の特徴が見えること
- 依頼主の自己申告と実態に小さなズレがあること
- 成功・失敗にかかわらず、次に残る余波があること

## 2. 序盤案件一覧

## 2.1 春の白麦平野: `春畦の水争い`

- 地域帯: 白麦平野
- 依頼カテゴリ: 交渉
- 依頼主カテゴリ: 公益依頼主
- 表向きの依頼内容:
- 隣接する二村の用水配分でもめており、春の種まき前に仲裁してほしい
- 実際の切迫点:
- 片方の村が先に境界杭を動かしており、その事実が露見すると面子が潰れる
- 伏せられた情報:
- 依頼主である村名主自身が不正を黙認している
- 主危険:
- 交渉決裂による小競り合い
- 副危険:
- 春の作業遅延による収穫減
- 新設ギルドに回る理由:
- 報酬が薄く、既存ギルドは嫌がる
- 成功条件:
- 播種前に暫定配分案をまとめる
- 部分成功条件:
- 暴力化だけは防ぐが、禍根は残る
- 失敗時の余波:
- 秋の収穫期に対立が再燃しやすくなる
- 次につながるフック:
- どちらかの村から、穀物護衛や失踪調査が追加で来る

## 2.2 春の葦底湿原: `初葦の採り路`

- 地域帯: 葦底湿原
- 依頼カテゴリ: 採取
- 依頼主カテゴリ: 生活依頼主
- 表向きの依頼内容:
- 施療院向けの初物薬草を確保して持ち帰ってほしい
- 実際の切迫点:
- 採取民どうしで採り路の優先権争いが起きている
- 伏せられた情報:
- 依頼主は他所の採り路を内々に使っている
- 主危険:
- 足場の悪さと湿地生物
- 副危険:
- 採取民との衝突
- 新設ギルドに回る理由:
- 小口で地味、かつ現地の顔が利かないギルドは嫌がる
- 成功条件:
- 必要量を確保し、現地の怒りを買いすぎず帰還する
- 部分成功条件:
- 素材は持ち帰るが、採取民との関係が悪化する
- 失敗時の余波:
- 施療院の素材不足が後日の市内案件に波及する
- 次につながるフック:
- 工房側から採取権確認の交渉依頼が来る

## 2.3 春のエンベルク近郊: `種籾の朝便`

- 地域帯: 白麦平野
- 依頼カテゴリ: 護衛
- 依頼主カテゴリ: 商業依頼主
- 表向きの依頼内容:
- 種籾と農具を近郊村まで運ぶ朝便を護衛してほしい
- 実際の切迫点:
- 荷主は納期よりも、便が止まったという悪評を恐れている
- 伏せられた情報:
- 荷の一部に契約外の高価な工具が混じっている
- 主危険:
- 街道脇の小規模盗賊
- 副危険:
- 泥濘による遅延
- 新設ギルドに回る理由:
- 小規模で単価が低く、大手は定期契約を優先する
- 成功条件:
- 朝便をその日のうちに通す
- 部分成功条件:
- 荷は届くが、追加荷の存在が露見する
- 失敗時の余波:
- 荷主が責任を押しつけ、商会筋の印象が悪くなる
- 次につながるフック:
- 同荷主から定期便の試験案件が来る

- 成功条件:
- 箱の行方と責任の所在を穏当に整理する
- 部分成功条件:
- 紛失品は戻るが、どこかの宿に不満が残る
- 失敗時の余波:
- 巡礼期の宿場連携が悪化する
- 次につながるフック:

- 表向きの依頼内容:

- 成功条件:
- 紛失原因を特定し、契約破綻を避ける
- 部分成功条件:
- 真相は掴むが、どちらかが遺恨を残す
- 失敗時の余波:
- 河運筋で「扱いの雑なギルド」という噂が立つ

## 2.6 夏のエンベルク市内: `灯河祭の迷い子`

- 地域帯: エンベルク市内
- 依頼カテゴリ: 調査
- 依頼主カテゴリ: 生活依頼主
- 表向きの依頼内容:
- 祭の人混みで見失った子どもを探してほしい
- 実際の切迫点:
- 保護者は迷子だけでなく、子どもが見たかもしれない場面を恐れている
- 伏せられた情報:
- 子どもは祭具運搬の不正を目撃している
- 主危険:
- 雑踏、誘拐未遂、祭りの混乱
- 副危険:
- 市参事会に知られたくない裏事情
- 新設ギルドに回る理由:
- 祭の警備本筋ではなく、私的で急ぎの困りごとだから
- 成功条件:
- 子どもを無事保護し、騒ぎを広げすぎない
- 部分成功条件:
- 子どもは見つかるが、不正の噂が広がる
- 失敗時の余波:
- 祭関連の不祥事が大きく燃える
- 次につながるフック:
- 市参事会か商会から祭後処理の依頼が来る

- 実際の切迫点:

- 地域帯: 燈架回廊
- 依頼カテゴリ: 交渉
- 依頼主カテゴリ: 信仰依頼主
- 表向きの依頼内容:
- 雪で巡礼客が減った宿場で、寄進帳の負担配分を調整してほしい
- 実際の切迫点:
- 宿場の一つが負担から抜けようとしており、巡礼導線が乱れかねない
- 伏せられた情報:
- 教会側も特定宿場を優遇している
- 主危険:
- 交渉決裂による導線破綻
- 副危険:

- 新設ギルドに回る理由:

- 部分成功条件:
- 導線は維持するが、どこかに明確な不満が残る
- 失敗時の余波:

- 次につながるフック:

## 3. 運用メモ

- 12件全部を初日から出す必要はない
- 序盤は `季節に合う案件4〜6件 + 年中案件2件` くらいが扱いやすい
- 同じ案件でも、依頼主の所属や伏せ情報を入れ替えることで再利用できる

## 4. 年中案件の差し込み枠

- 市内失踪
- 倉庫紛失
- 小規模護衛
- 教会からの内密調査
- 採取民の揉め事
- 競合ギルドが断った討伐
