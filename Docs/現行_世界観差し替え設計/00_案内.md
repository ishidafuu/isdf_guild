# 現行 世界観差し替え設計 案内

最終更新: 2026-03-10

このディレクトリは、現在進行中の「世界観差し替え型ギルドゲーム」の共通フレームワーク設計を置く正本です。
今後の仕様整理、質問反映、AI 実装前の詰めはここを基準に行います。

## 目的

- コアループと世界観レイヤーを分離して整理する
- 世界観パック追加や自動生成に耐える共通仕様を保持する
- 旧来の中世ファンタジー文書と混線させず、再利用可能な設計だけを蓄積する
