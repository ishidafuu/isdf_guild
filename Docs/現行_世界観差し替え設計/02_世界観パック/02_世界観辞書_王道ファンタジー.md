# 世界観辞書 王道ファンタジー

最終更新: 2026-03-10

## 1. 目的

本章は、共通依頼構造や共通キャラ構造に対して、王道ファンタジー世界で使う語彙辞書を定義する。
処理は共通、表示と演出だけを差し替える。

## 2. 能力値の表示名

- `power`: 武勇
- `tech`: 技巧
- `sense`: 察知
- `social`: 話術
- `will`: 信念

## 3. ロール名

- `frontliner`: 剣士 / 騎士
- `support`: 神官 / 助祭 / 支援魔術師
- `scout`: 斥候 / 狩人 / 野伏
- `engineer`: 工匠 / 錬金術師 / 魔導技師
- `negotiator`: 交渉人 / 吟遊詩人 / 口利き

## 4. 依頼カテゴリ表示名

- `subjugation`: 魔物討伐 / 害獣退治 / 異形鎮圧
- `escort`: 隊商護衛 / 巡礼護衛 / 旅路警護
- `investigation`: 異変調査 / 行方追跡 / 真相探り
- `recovery`: 遺物回収 / 呪具奪還 / 資材確保
- `delivery`: 物資運搬 / 密使輸送 / 供物搬送
- `negotiation`: 領主折衝 / 商談仲裁 / 慣習調停
- `defense`: 村落防衛 / 関所死守 / 結界守護
- `rescue`: 遭難者救出 / 捕縛者奪還 / 迷い人捜索

## 8. 障害辞書

- 夜になると霧が濃くなる
- 地元住民が立ち入りを嫌がる
- 聖域ゆえに勝手な破壊ができない
- 対象が火を恐れず襲ってくる
- 領主と神殿の利害が衝突している
- 橋が壊れて正規ルートが使えない
- 依頼人が重要情報を伏せている
- 競合冒険者が先行している
- 護送対象がわがまま
- 遺物に呪いが残っている

## 11. 特技名の言い換え例

- 戦術打撃: 渾身斬り / 破邪の一撃
- 応急処置: 応急治療 / 祈祷治癒
- 痕跡読解: 狩人の目 / 足跡見
- 交渉術: 話術 / 宮廷作法
- 封印処理: 呪具封じ / 護符封緘

## 12. 欠点名の言い換え例

- 短気: 血気盛ん
- 過剰慎重: 腰が重い
- 名誉優先: 面子第一
- 臆病: 化け物嫌い
- 疑い深い: 人間不信
- お人好し: 世話焼き

## 13. 依頼文テンプレ差し替え例

- 調査:
  - 近ごろ `場所` にて `異変` が続いており、原因を探ってほしい
- 護衛:
  - `護衛対象` を `目的地` まで無事に送り届けてほしい
- 回収:
  - `場所` に眠る `回収対象` を、損なわず持ち帰ってほしい

## 14. トーンガイド

- 文体はやや格調高くしつつ、庶民依頼では素朴さを残す
- 名誉、信仰、慣習、土地の記憶をにじませる
- 戦闘だけでなく、土地ごとのしがらみを依頼に混ぜる
