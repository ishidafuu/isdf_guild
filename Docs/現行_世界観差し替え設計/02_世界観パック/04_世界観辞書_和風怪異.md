# 世界観辞書 和風怪異

最終更新: 2026-03-10

## 1. 目的

本章は、共通依頼構造や共通キャラ構造に対して、和風怪異世界で使う語彙辞書を定義する。

## 2. 能力値の表示名

- `power`: 武
- `tech`: 手際
- `sense`: 見立て
- `social`: 口利き
- `will`: 心胆

## 3. ロール名

- `frontliner`: 用心棒 / 祓い手
- `support`: 巫女 / 法師 / 世話役
- `scout`: 見聞役 / 山追い / 先触れ
- `engineer`: 結界師 / 封具師 / 細工師
- `negotiator`: 口利き / 調停役 / 顔役

## 4. 依頼カテゴリ表示名

- `subjugation`: 怪異鎮め / 祟り祓い / 荒神退け
- `escort`: 身辺守護 / 行列守り / 封具護送
- `investigation`: 異変聞き取り / 由来調べ / 神隠し追い
- `recovery`: 呪物回収 / 封具取り戻し / 依代確保
- `delivery`: 封具運び / 奉納搬送 / 急ぎ届け
- `negotiation`: 土地神調停 / 村境仲裁 / 因習折り合い
- `defense`: 結界守り / 村守り / 封印持ちこたえ
- `rescue`: 神隠し救出 / 迷い人連れ戻し / 身請け奪還

## 8. 障害辞書

- 誰も本当のことを語ろうとしない
- 古い約定のせいで手順を省けない
- 怪異を傷つけると祟りが広がる
- 村の外の者を嫌う空気が強い
- 夜しか手掛かりが現れない
- 土地の境を越えると力が乱れる
- 名家が面子を守ろうとする
- 依頼人自身が因果に巻かれている
- 封具の扱いを誤ると被害が増える
- 人と怪異のどちらにも言い分がある

## 11. 特技名の言い換え例

- 戦術打撃: 祓い打ち / 崩し太刀
- 応急処置: 手当て / 清め
- 痕跡読解: 気配読み / 由来見
- 交渉術: 口利き / 話まとめ
- 封印処理: 結び直し / 封じ手

## 12. 欠点名の言い換え例

- 短気: 気が立ちやすい
- 過剰慎重: 腰が引ける
- 名誉優先: 家名にこだわる
- 臆病: 夜道が駄目
- 疑い深い: よそ者嫌い
- お人好し: 情に流される

## 13. 依頼文テンプレ差し替え例

- 調査:
  - `場所` で続く `異変` のわけを見立ててほしい
- 護衛:
  - `護衛対象` を `目的地` まで穏便にお守りいただきたい
- 回収:
  - `場所` にある `回収対象` を、荒立てず持ち帰ってほしい

## 14. トーンガイド

- 文体は静かで余白を持たせる
- 露骨な説明よりも、因習や気配を匂わせる
- 単純な善悪ではなく、土地・家・怪異の事情を並べる
