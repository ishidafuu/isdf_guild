# Docs 案内

最終更新: 2026-03-10

このディレクトリは、ギルドゲーム構想に関する文書の入口です。
現在は「今使う設計」と「いったん退避した旧世界観案」を分けて管理します。

## いま見る場所

- `現行_世界観差し替え設計/`
  - 現在の正本です。世界観差し替え型ギルドゲームとして詰めている共通仕様を置きます。
- `退避_旧世界観案/`
  - 以前の中世ファンタジー寄りの世界観案です。削除はせず、参照用に退避しています。

## ディレクトリ方針

- `00_` は入口や案内
- `01_` 以降はテーマ別のまとまり
- `99_` は更新履歴
