# 世界観差し替え型ギルドゲーム 企画書 1ページ版

最終更新: 2026-03-10

## タイトル仮称

世界観差し替え型ギルドゲーム

## 概要

本企画は、ギルド運営と依頼解決をコアに据えたゲームである。
プレイヤーはギルドに所属するメンバーを運用し、依頼の達成によって資源・評価・拠点機能を獲得しながら組織を成長させる。
特徴は、コアルールを共通化しつつ、世界観のみを差し替え可能な構造を持つ点にある。

## コンセプト

「同じギルド運営ゲームを、異なる世界の皮膚で遊ぶ」

## コア体験

- 依頼を選ぶ
- 仲間と準備する
- 判定で任務を解決する
- 成功や失敗の結果を受ける
- 報酬でキャラとギルドを成長させる

## メインループ

1. 依頼提示
2. 依頼選択
3. 準備
4. 任務遂行
5. 判定解決
6. 結果反映
7. 報酬獲得
8. 成長・拠点更新

## 本作の特徴

- ルール本体と世界観レイヤーを分離する
- 世界観ごとに依頼、敵、報酬、勢力、拠点の見え方が変わる
- 同じルールでも異なる物語体験を提供できる
- 追加ワールドや自動生成との相性が良い

## 固定するもの

- 判定構造
- 能力値の機能
- ロールの機能
- 依頼カテゴリの機能
- 成長とギルド運営の進行構造

## 差し替えるもの

- 用語
- 依頼文
- 敵カテゴリ
- 拠点外観
- 報酬名
- 勢力名
- イベント表
- 文体と雰囲気

## 想定されるワールドパック例

- 王道ファンタジー
- 退廃サイバーパンク
- 和風怪異
- 宇宙辺境開拓
- 終末交易共同体

## 拡張性

本作は単一世界観の作品ではなく、「ギルドゲームのコアシステム」に複数の世界観を載せるフレームワークとして展開可能。
追加パック、人力制作、半自動生成のいずれにも対応しやすい。
