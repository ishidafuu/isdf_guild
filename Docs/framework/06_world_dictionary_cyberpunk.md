# 世界観辞書 退廃サイバーパンク

最終更新: 2026-03-10

## 1. 目的

本章は、共通依頼構造や共通キャラ構造に対して、退廃サイバーパンク世界で使う語彙辞書を定義する。

## 2. 能力値の表示名

- `power`: 制圧
- `tech`: 技術
- `sense`: 走査
- `social`: 交渉
- `will`: 耐性

## 3. ロール名

- `frontliner`: 制圧オペレーター / エンフォーサー
- `support`: メディック / バッファー / サポートランナー
- `scout`: トレーサー / スカウト / ウォッチャー
- `engineer`: ハッカー / メカニック / システム技師
- `negotiator`: ブローカー / ネゴシエーター / フィクサー

## 4. 依頼カテゴリ表示名

- `subjugation`: 排除任務 / 制圧依頼 / 暴走停止
- `escort`: VIP護送 / 警護搬送 / 逃走補助
- `investigation`: 追跡調査 / ログ解析 / 情報洗浄
- `recovery`: データ奪取 / 現物確保 / 試作品回収
- `delivery`: 密輸搬送 / ブラック便 / 緊急配送
- `negotiation`: 取引仲裁 / 停戦調整 / 権利交渉
- `defense`: アジト防衛 / ノード防衛 / 区画保持
- `rescue`: 身柄奪還 / 脱出支援 / 生存者回収

- 下層区の住民代表
- 独立運び屋

- 匿名依頼人
- 地区自警団
- ブローカー
- ハッカーセル

- 下層居住区
- 路地裏マーケット
- 廃工場区画
- 高架下の密輸ルート
- データセンター跡
- 企業ビルの保守階
- 地下医療施設
- 検問ゲート周辺
- 無人倉庫街
- 旧交通網
- ドローンドック
- 監視塔の死角

## 7. 脅威辞書

- 企業警備部隊
- 追跡ドローン
- 違法改造兵
- 監視AI
- ストリートギャング
- 暴走メンテナンス機
- 情報汚染ウイルス
- 自律砲台
- 身元不明の暗殺者
- 検問部隊
- 企業の回収班
- 神経汚染された群衆

## 8. 障害辞書

- 監視網が張り巡らされている
- 認証が偽装できない
- 輸送物の中身が不安定
- 地区封鎖が始まる
- 通信が傍受されている
- 依頼人が契約条件を伏せている
- 敵より先に企業監査が来る
- 内通者の疑いがある
- データが分割されている
- 現場の電力が不安定

## 11. 特技名の言い換え例

- 戦術打撃: 制圧射撃 / バーストブレイク
- 応急処置: 緊急安定化 / ナノパッチ
- 痕跡読解: ログ追跡 / シグナル解析
- 交渉術: 取引術 / 圧力調整
- 封印処理: 隔離処理 / ブラックボックス封緘

## 12. 欠点名の言い換え例

- 短気: キレやすい
- 過剰慎重: ハイリスク忌避
- 名誉優先: 筋を通しすぎる
- 臆病: 追跡恐怖
- 疑い深い: パラノイア気味
- お人好し: カモ体質

## 13. 依頼文テンプレ差し替え例

- 調査:
  - `場所` で発生している `異変` の発信源を洗ってほしい
- 護衛:
  - `護衛対象` を `目的地` までノイズなく通してほしい
- 回収:
  - `場所` に残された `回収対象` を痕跡なしで確保してほしい

## 14. トーンガイド

- 文体は乾いていて即物的、説明は短く切る
- 企業、契約、監視、損耗、都市の冷たさを前面に出す
- 金だけでなく、アクセス権やルートの価値を強調する
