# 次に確認したい質問事項

最終更新: 2026-03-09

## 0. この章の目的

- ここまでの世界観構築を踏まえて、今後ユーザーに確認したい論点をまとめる
- 優先度の高いものから順に確認できるようにする
- 好みの問題と仕様固定の問題を分けて整理する

## 1. 優先度高

### 1.1 教会の最終温度感

- 現在は「公共性もあるが、体面や権威も重い」中間値で置いている
- このまま中庸でよいか、もう少し善寄りか、面倒寄りかを確認したい

### 1.2 遺跡の最終トーン

- 現在は「神秘と実用品の中間」「完全SFにはしない」で進めている
- もう少し宗教的神秘を強めるか、古代技術感を強めるかを確認したい

### 1.3 他ギルドとの関係の強度

- 現在は競争中心で、一部に敵対や警戒が混じる前提
- 業界的ライバル関係を主にするか、もっと強い対立も増やすかを確認したい

## 2. 優先度中

### 2.1 教会・商会・市参事会の三すくみの濃さ

- いまは三者の力関係がある前提だが、政治劇としてどこまで前に出すかは未固定

### 2.2 異常系危険生物の濃さ

- 現状は獣害、湿地、水辺、遺構系が主
- もっと不気味さや信仰解釈の余地を増やすかを確認したい

### 2.3 依頼主の「嫌な感じ」の強さ

- 不完全情報や責任逃れは入れているが、どの程度まで嫌な人間を出してよいかの好みを確認したい

### 2.4 冒険者の生活苦の重さ

- 金策や生活事情は入れているが、全体の空気を暗くしすぎないラインを確認したい

## 3. 優先度低

### 3.1 都市外のさらに広い世界

- 今はエンベルク都市圏に集中している
- 遠方王権、他都市、広域政治をどこまで作るかは後回しでよいが、最終的には確認したい

### 3.2 祭事の具体数

- いまは `灯河祭` を置いている
- 季節イベントを増やすかは後で判断可能

### 3.3 プレイヤーギルド拠点の日常描写量

- 現在は依頼と面談中心で、生活描写は背景寄り
- どこまで拠点の生活感を増やすかは実装フェーズで再確認したい

## 4. 今後こちらから提案しやすいテーマ

- 他ギルド別の運営方針テンプレート
- 依頼主の満足・不満・再依頼文のテンプレート
- 冒険者自己紹介テンプレート
- 中盤案件プール
- 地域顔役テンプレートの具体化
