# 危険生物テンプレート

最終更新: 2026-03-09

## 0. この章の目的

- エンベルク圏の危険生物を、地域文脈と依頼文脈に結びつけて整理する
- モンスターが単なる戦闘要員にならないよう、発生理由と被害構造を定義する
- AI生成や手動案件設計で再利用しやすいテンプレート単位を用意する

## 1. 基本方針

- 危険生物は「強い弱い」だけでなく、「どこにいるか」「何を荒らすか」「なぜ今問題化したか」で扱う
- 同じ討伐依頼でも、完全駆除より追い払い、航路確保、巣の移動確認など目的を分ける
- 危険生物は土地や季節と結びつき、地域ごとに違う意味を持つ
- 人為や自然危険を打ち消さないよう、生物由来の脅威は全体の一部として配置する

## 2. 分類軸

危険生物は大きく5系統で扱う。

### 2.1 獣系

- もともと自然界に近いが、人里被害が増えると依頼化する
- 白麦平野、燈架回廊で使いやすい

### 2.2 湿地系

- 葦底湿原の足場、水路、採取路と結びつく
- 単純討伐より「通れるようにする」目的が多い

### 2.3 遺構系

- 灰冠丘陵の旧祈祷施設や遺跡周辺に現れる
- 発掘や侵入など、人為がきっかけになりやすい

### 2.4 水辺系

- 舟骨水道や河岸の荷役と結びつく
- 事故や密輸と重なると、被害の切り分けが難しい

### 2.5 異常系

- 教会や地元民に「祟り」「徴」と呼ばれやすい
- 実態が生物か現象か曖昧なものも含む
- 調査から討伐、交渉へもつなぎやすい

## 3. 地方帯別テンプレート

## 3.1 白麦平野

### `畦喰い牙猪`

- 系統: 獣系
- 概要:
- 用水路と畦を荒らす大型猪。収穫より苗床を壊すため恨みを買いやすい
- よく出る時期:
- 春、秋
- 典型被害:
- 畦崩し、家畜驚乱、夜間の荷車転倒
- 依頼化のされ方:
- 討伐、追い払い、護衛便の迂回警戒
- 誤認・ズレ:
- 「魔物化した」と誇張されやすいが、実際には餌場変化が原因のことも多い
- フック:
- 井戸や飼葉置き場の管理不全、密猟、村境争いと接続しやすい

### `灰尾の追い犬`

- 系統: 獣系
- 概要:
- 野犬化した群れ。荷車の端や負傷家畜を狙う
- よく出る時期:
- 冬、初春
- 典型被害:
- 家畜被害、街道沿いの遅れた便への追撃
- 依頼化のされ方:
- 討伐、夜間護衛、群れ追跡
- 誤認・ズレ:
- 夜盗と被害が混同されることがある
- フック:
- 冬の備蓄不足、市内廃棄場管理、 abandoned hamlets と結びつく

## 3.2 葦底湿原

### `泥腹の大蜥`

- 系統: 湿地系
- 概要:
- 渡し場や浅瀬に居つく大型蜥蜴。縄張りに近づく小舟を転覆させる
- よく出る時期:
- 夏、秋
- 典型被害:
- 渡し場停止、採取路寸断、小舟事故
- 依頼化のされ方:
- 討伐、追い払い、迂回路確保
- 誤認・ズレ:
- 「瘴気の獣」と呼ばれるが、巣に近づいた人間側の事情も大きい
- フック:
- 蜂蜜採取、魚醤仕込み、違法採取の拡張と結びつく

### `葦影の刺嘴`

- 系統: 湿地系
- 概要:
- 人を積極的には狙わないが、群れると採取路の安全を崩す大型鳥
- よく出る時期:
- 春、夏
- 典型被害:
- 採取民の負傷、荷傷み、視界妨害
- 依頼化のされ方:
- 採取護衛、間引き、巣の位置調査
- 誤認・ズレ:
- 毒を持つと恐れられるが、実際には嘴傷と感染が主因
- フック:
- 施療院素材不足、採取権争い、外来業者の参入と相性がよい

## 3.3 灰冠丘陵

### `塔守りの痩獣`

- 系統: 遺構系
- 概要:
- 旧祈祷塔周辺に現れる異常に痩せた四足獣。光や音に過敏
- よく出る時期:
- 通年
- 典型被害:
- 調査隊襲撃、夜営妨害、遺物搬送時の事故
- 依頼化のされ方:
- 討伐、護衛、巣の特定、誘導離脱
- 誤認・ズレ:
- 遺構そのものの呪いと見なされるが、無許可発掘後に活発化しやすい
- フック:
- 学匠院への不信、禁区侵入、地元案内人の警告無視と結びつく

### `石背の掘り喰い`

- 系統: 遺構系
- 概要:
- 崩れかけた石積みの隙間に巣を作る小型群生生物
- よく出る時期:
- 通年、特に乾季
- 典型被害:
- 崩落誘発、荷傷み、脚周りの負傷
- 依頼化のされ方:
- 調査護衛、巣の処理、運搬路整備
- 誤認・ズレ:
- 危険度は低く見えるが、集団事故の引き金になりやすい
- フック:
- 小規模案件から大きな崩落事故へつなぎやすい

## 3.4 舟骨水道

### `河腹の裂吻魚`

- 系統: 水辺系
- 概要:
- 荷落ちや血の臭いに寄りやすい大型魚。浅瀬で小舟を揺らす
- よく出る時期:
- 夏、河止め前後
- 典型被害:
- 荷の回収困難、小舟転覆、渡しの停止
- 依頼化のされ方:
- 河路警戒、漁場変更調査、荷回収補助
- 誤認・ズレ:
- 河賊被害と混ざって話が大きくなりやすい
- フック:
- 密輸荷の投棄、傷病人の搬送失敗、偽標識と絡められる

### `岸這い水獣`

- 系統: 水辺系
- 概要:
- 河岸の倉庫地帯近くに上がる半水棲獣。夜間の騒ぎを起こす
- よく出る時期:
- 夏、初秋
- 典型被害:
- 夜番負傷、荷崩し、家畜驚乱
- 依頼化のされ方:
- 夜警護衛、追い払い、巣の位置調査
- 誤認・ズレ:
- 内部犯の荷抜きを隠す口実に使われることがある
- フック:
- 倉庫番の不正、夜荷の数え違い案件と相性が良い

## 3.5 燈架回廊

### `灯盗り羽虫`

- 系統: 異常系
- 概要:
- 灯火に異様に集まり、宿場の夜警や巡礼灯を乱す群飛虫
- よく出る時期:
- 夏、秋
- 典型被害:
- 灯火管理の失敗、宿場混乱、夜道事故
- 依頼化のされ方:
- 原因調査、灯路警備、巣の除去
- 誤認・ズレ:
- 不吉の徴と騒がれるが、実際には周辺の香油保管や水場条件が影響する
- フック:
- 寄進品管理、偽巡礼、宿場間対立に接続する

### `巡礼喰いの痩狼`

- 系統: 獣系
- 概要:
- 巡礼団の後尾や病人を狙う痩せ狼。単体より群れ連携が危険
- よく出る時期:
- 冬、巡礼期直前
- 典型被害:
- 後尾崩し、夜営襲撃、護衛の疲弊
- 依頼化のされ方:
- 巡礼護衛、討伐、追跡
- 誤認・ズレ:
- 教義上の「罰」と解釈されやすく、現実的対処が遅れることがある
- フック:
- 教会の面子、宿場負担、体調不良者の切り捨て問題と絡めやすい

## 4. 共通の依頼化テンプレート

危険生物を依頼へ落とすときは、次の流れで使う。

- `生物名`
- `地域帯`
- `今問題化した理由`
- `誰に被害が出ているか`
- `依頼主は何を望んでいるか`
- `本当に必要な対応は何か`
- `誤認や誇張は何か`
- `討伐後に何が残るか`

## 5. 強さではなく扱いで差を出すルール

- 同じ生物でも、孤立個体か群れかで案件の意味を変える
- 「完全討伐」以外の解決手段を必ず候補に置く
- 発生原因に人間側の行動が絡む余地を持たせる
- 生物名だけで恐怖を作らず、被害様式で印象を作る
