# 冒険者出自テンプレート

最終更新: 2026-03-09

## 0. この章の目的

- エンベルクへ流入する冒険者の出自を、世界観に沿って整理する
- 冒険者が単なる戦力でなく、仕事の選び方や依頼主との相性を持つ理由を明文化する
- 自動生成と手動設計の両方に使える母集団テンプレートを用意する

## 1. 基本方針

- 冒険者は英雄階級ではなく、危険仕事を請ける自由業者の総称
- 出自の違いは、能力よりも `何を嫌うか / 何を誇るか / 何に弱いか` に反映する
- エンベルクへ来る理由が地域構造や景気と結びついていることを重視する
- 同じ出自でも人格を固定せず、あくまで生成の骨格として扱う

## 2. 出自テンプレート

## 2.1 街道警備あがり

- 概要:
- かつて隊商や関所で警備に従事していた実務派
- 好む仕事:
- 護衛、運搬、退路確保、定期便
- 苦手仕事:
- 華を求める見世物的依頼、曖昧な調査
- 誇り:
- 無事に通すこと
- 反発点:
- 危険説明の不足、無理な強行
- エンベルクに来る理由:
- 再開街道時代で民間護衛の需要が増えたため

## 2.2 村猟師・山案内

- 概要:
- 地元の野山や湿地に詳しい案内兼狩人
- 好む仕事:
- 調査、採取、追跡、獣害対処
- 苦手仕事:
- 都市内交渉、格式張った護衛
- 誇り:
- 土地勘と見切り
- 反発点:
- 都市人の机上判断
- エンベルクに来る理由:
- 季節仕事だけでは食えず、都市の依頼市場に流れてくる

## 2.3 兵役崩れ

- 概要:
- 正規兵や徴募兵を離れた者
- 好む仕事:
- 討伐、護衛、即応案件
- 苦手仕事:
- 長い駆け引き、学術調査
- 誇り:
- 規律、反応速度、隊列維持
- 反発点:
- 指揮系統の曖昧さ
- エンベルクに来る理由:
- 盟約都市圏の民間案件が増え、腕を売れる場がある

## 2.4 没落家筋・下級騎士

- 概要:
- 体面はあるが資金や地位を失い、民間仕事に降りてきた層
- 好む仕事:
- 名誉の立つ護衛、教会案件、見栄えのよい討伐
- 苦手仕事:
- 汚れ仕事、採取、荷役色の強い運搬
- 誇り:
- 名誉、約定、立ち居振る舞い
- 反発点:
- 露骨な金勘定、不名誉な扱い
- エンベルクに来る理由:
- 巡礼路と商路があり、体面を保ちながら仕事を選べる

## 2.5 学匠院くずれ

- 概要:
- 学匠院関係者、筆写師、助手、見習い崩れ
- 好む仕事:
- 調査、回収、資料護送、遺跡案件
- 苦手仕事:
- 単純な力押し、泥臭い護衛
- 誇り:
- 知識、記録、観察眼
- 反発点:
- 対象を雑に扱うこと
- エンベルクに来る理由:
- 灰冠丘陵の遺構需要と、正規雇用の不足

## 2.6 巡礼護衛・教会雑務あがり

- 概要:
- 巡礼路や施療院、末寺で俗務を担っていた者
- 好む仕事:
- 巡礼護衛、失踪捜索、教会案件、生活支援寄り依頼
- 苦手仕事:
- あからさまな非公式依頼
- 誇り:
- 人を無事に運ぶこと、筋を通すこと
- 反発点:
- 信仰や弱者を食い物にする態度
- エンベルクに来る理由:
- 巡礼期の季節波が大きく、通年の仕事を求めて都市へ残る

## 2.7 河船勤めあがり

- 概要:
- 船頭、荷役、河岸宿の用心棒など水運筋の経験者
- 好む仕事:
- 河運護衛、夜警、帳尻の合う運搬
- 苦手仕事:
- 山道調査、遺跡深部
- 誇り:
- 時刻感覚、荷扱い、河筋の知識
- 反発点:
- 書付なしの口約束
- エンベルクに来る理由:
- 河運が都市経済の動脈であり、民間依頼化しやすい

## 2.8 旅芸人・見世物育ち

- 概要:
- 一座、芸人、呼び込み、軽業などの流れ者出身
- 好む仕事:
- 潜入、撹乱、祭り案件、見られる依頼
- 苦手仕事:
- 地味な定期護衛、長い待機
- 誇り:
- 人前で場を動かすこと
- 反発点:
- 軽んじられること、見せ場を奪われること
- エンベルクに来る理由:
- 祭事、巡礼、市場があり、仕事口も観客もいる

## 2.9 流れ採取民

- 概要:
- 湿地、山野、薬草地帯を渡り歩いてきた素材屋
- 好む仕事:
- 採取、調査、裏道案内
- 苦手仕事:
- 規律の強い隊列運用、都市礼法
- 誇り:
- 旬を読むこと、危険域の勘
- 反発点:
- 権利を知らない外部者の横入り
- エンベルクに来る理由:
- 工房と施療院が素材を買うため

## 2.10 敗残傭兵・半端者

- 概要:
- どこにも定着できず、実力も性格も不安定な層
- 好む仕事:
- 即金性の高い案件、危険高報酬案件
- 苦手仕事:
- 継続契約、丁寧な対人調整
- 誇り:
- 生き延びてきたことそれ自体
- 反発点:
- 見下されること、公正を欠く配分
- エンベルクに来る理由:
- 新設ギルドや雑多な都市圏が受け皿になりやすい

## 3. 出自と季節の相性

- 収穫期:
- 街道警備あがり、河船勤めあがりが強い

- 巡礼期:
- 巡礼護衛あがり、没落家筋、旅芸人が動きやすい

- 冬:
- 兵役崩れ、街道警備あがり、生活苦の半端者が出やすい

- 灰冠丘陵の調査期:
- 学匠院くずれ、山案内が強い

## 4. ギルド運営との接続

- 同じギルドでも、どの出自が集まりやすいかで空気が変わる
- 安全重視なら街道警備あがりや巡礼護衛あがりが定着しやすい
- 変則案件が多いと旅芸人や半端者が流入しやすい
- 学術案件が多いと学匠院くずれが増え、教会案件が多いと巡礼護衛あがりが残りやすい

## 5. 生成用の最低項目

- `誇り`
- `嫌う扱い`
- `向く依頼`

- `今困っている生活事情`
