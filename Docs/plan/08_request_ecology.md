# ギルド周辺環境・依頼生態系

最終更新: 2026-03-09

## 0. この章の目的

- プレイヤーギルドの周囲にいる依頼主、競合ギルド、地域顔役の構造を固定する
- 依頼が「どこから、なぜ、どういう温度感で」流れ込むかを明文化する
- ミッションや会話生成で使える役割テンプレートを用意する

## 1. 依頼生態系の考え方

- エンベルクでは依頼は無から生まれず、必ず `誰かの利害` から発生する
- 依頼は大きく `商業`, `公益`, `信仰`, `生活`, `研究`, `非公式処理` の6系統に分かれる
- 新設ギルドは、最初から大口案件の中心に入るのではなく、既存ギルドが拾いきれない案件から信用を積む
- したがって、世界観側で「新設ギルドに流れやすい案件の性質」を明示しておく

## 2. 依頼主カテゴリ

## 2.1 商業依頼主

### 主な所属

- ヴァルド商業連盟支部
- 中小商会
- 倉庫業者
- 河船主
- 旅籠主

### 典型的な困りごと

- 納期遅延
- 積荷紛失
- 定期便の護衛不足
- 取引先との責任押しつけ合い
- 秘密裏に処理したい不祥事

### 依頼の特徴

- 報酬は比較的良い
- 情報は必要な分だけしか出さない傾向がある
- 継続契約になりやすい
- 収益性重視の評判に強く反応する

## 2.2 公益依頼主

### 主な所属

- 市参事会下役
- 地方代官
- 宿場の顔役
- 村の名主

### 典型的な困りごと

- 治安維持の手足不足
- 村同士の争い
- 公に大ごとにしたくない揉め事
- 予算不足の中での異変対応

### 依頼の特徴

- 報酬は高くない
- 正当性や説明責任が重い
- 失敗時の風評が広がりやすい
- 適切性と倫理性の評判に強く反応する

## 2.3 信仰依頼主

### 主な所属

- 聖火教会の司祭、助祭、巡礼宿の世話役
- 施療院の管理係
- 巡礼団の取りまとめ役

### 典型的な困りごと

- 巡礼路の安全確保
- 行方不明巡礼者
- 寄進品や聖具の紛失
- 教会内部では処理しにくい対外案件

### 依頼の特徴

- 体面を重んじる
- 依頼内容が婉曲になりやすい
- 聖地や遺構を巡る解釈差が火種になる
- 倫理性の評判に特に敏感

## 2.4 生活依頼主

### 主な所属

- 個人商人
- 市民
- 農家
- 採取民
- 小さな宿や工房

### 典型的な困りごと

- 家族の失踪
- 家畜被害
- 小規模な盗難
- 採取路の危険
- 近隣とのこじれ

### 依頼の特徴

- 小口案件が多い
- 感情表現が直接的
- 誇張や思い込みも混じりやすい
- 新設ギルドの最初の評判形成に直結する

## 2.5 研究依頼主

### 主な所属

- ローデン学匠院分室
- 個人学者
- 収集家
- 筆写師

### 典型的な困りごと

- 調査隊不足
- 回収資料の護送
- 研究成果の奪取懸念
- 危険地帯への同行者不足

### 依頼の特徴

- 報酬や学術価値は高いが、危険認識が甘い
- 説明は細かいが、現場の危険が抜けることがある
- 調査、採取、運搬に強く結びつく

## 2.6 非公式依頼主

### 主な所属

- 身元を明かしたがらない荷主
- 顔役の私的使者
- 教会や商会の下請け口

### 典型的な困りごと

- 表沙汰にできない不始末
- 契約前の下調べ
- 失踪や裏切りの確認

### 依頼の特徴

- 高報酬か極端な説明不足になりやすい
- 失敗時の倫理性リスクが高い
- 新設ギルドが安易に寄りすぎると色がつきやすい

## 3. 競合ギルド類型

## 3.1 `赤鉤団`

- 類型: 荒事即応型
- 強み: 討伐、強襲、短期決戦
- 弱み: 長期護衛、丁寧な説明、冒険者消耗
- 依頼主からの見え方: 速いが荒い
- 冒険者からの見え方: 稼げるが危ない
- 新設ギルドとの関係: 危険高報酬案件で競合しやすい

## 3.2 `白樽組合`

- 類型: 定期便堅実型
- 強み: 運搬、河運、定期護衛
- 弱み: 変則案件、遺跡、柔軟対応
- 依頼主からの見え方: 安定するが融通が利かない
- 冒険者からの見え方: 安全だが地味
- 新設ギルドとの関係: 小回りの利く商業案件で差別化余地がある

## 3.3 `灯章会`

- 類型: 教会近接型
- 強み: 巡礼護衛、聖地周辺、対教会調整
- 弱み: 俗っぽい案件、不名誉案件
- 依頼主からの見え方: 品位はあるが気軽に頼みにくい
- 冒険者からの見え方: 筋はいいが堅苦しい
- 新設ギルドとの関係: 教会案件のこぼれや民間寄り案件で競合する

## 4. 新設ギルドに流れやすい案件

- 小口すぎて大手が拾わない
- 継続案件だが儲けが薄い
- 情報が汚く、既存ギルドが嫌がる
- 地域顔役との関係がまだ薄く、様子見で回される
- 急ぎだが規模は小さい
- 表看板にはしづらいが、誰かが処理しないと困る

これにより、序盤の案件が「都合よく簡単だから来る」のではなく、「他が取りにくいから来る」と説明できる。

各地方帯には、固定NPCではなくてもよいので、次の役割が存在する前提にする。

### 灰冠丘陵

- 山道案内人
- 古物買い付け人
- 立入を嫌う集落の長

### 白麦平野

- 村名主
- 用水番
- 巡礼宿の女将

### 葦底湿原

- 採取頭
- 施療院への仲買人
- 小舟渡し

### 舟骨水道

- 船主
- 倉庫番頭
- 河岸宿の帳場役

### 燈架回廊

- 巡礼宿世話役
- 末寺の司祭
- 村の古老

## 6. 依頼主生成の基礎単位

自動生成時は、少なくとも次の組み合わせで依頼主を定義する。

- `所属地域`
- `所属組織`
- `役割`
- `現在の困りごと`
- `守りたいもの`
- `隠したいこと`

例:

- `白麦平野 / 村 / 名主 / 用水争いが悪化 / 今年の収穫 / 先に境界杭を動かしたこと`
- `舟骨水道 / 商会 / 倉庫番頭 / 荷が一つ消えた / 取引先との信用 / 帳簿の書き換え`

## 7. プレイヤーギルドの周囲にいる人々

プレイヤーギルドの「周囲」として、常に次の層が存在すると考える。

- 持ち込み依頼主
- 案件を回すか迷っている中間管理役
- 他ギルドへ行くか比較している依頼主
- 応募や保留をする冒険者
- 噂を広める街の観測者

これにより、ギルドは単なる掲示板ではなく、街の利害が交差する窓口になる。

## 8. ミッション内容へ落とす際のルール

- 商業依頼は「納期」「秘密」「責任押しつけ」のいずれかを混ぜる
- 公益依頼は「予算不足」「面子」「被害の広がり」を混ぜる
- 信仰依頼は「建前と本音の差」を混ぜる
- 生活依頼は「感情の切実さ」と「情報の曖昧さ」を混ぜる
- 研究依頼は「説明の細かさ」と「危険認識の甘さ」を混ぜる
