# キャラ生成サンプル（試作2名）

最終更新: 2026-03-09

このファイルは、`23_ai_generation_design.md` で定義した `キャラ調査書` の試作例です。
MVPでは、以下のような `判定用骨格 + 表示用要約 + 内部調査書` を1セットで持つ想定です。

## サンプル1: エッダ・グラウベル

### 判定用骨格

- 役割: 冒険者
- 通り名: 灰帳のエッダ
- 年齢帯: 20代後半
- BIG5: `O=38 / C=82 / E=34 / A=61 / N=57`
- background_tags: `元街道警備`, `家計負担`, `帳簿癖`
- skill_tags: `槍`, `護衛`, `地理勘`, `交渉補佐`
- self_eval_bias: `慎重寄り`
- 初期傾向:
- 地味な護衛や運搬を過小評価しない
- 依頼主の説明不足に強く反応する
- 一度信頼した相手には長く付き合う

### public_digest

几帳面で口数は少ないが、危険の見積もりに関しては誰よりもしつこい槍手。無茶な英雄譚より、全員が帰る仕事を良しとする。

### volatile_hook

今月の仕送り額が足りず、割の悪い依頼に苛立っている。

### private_dossier

- core_identity:
- 王都近郊の街道警備隊あがり。身なりは質素だが道具の手入れだけは徹底している。
- 初対面では冷たく見えるが、実際は「失う前に先回りする」癖が強いだけ。

- daily_texture:
- 早起きで、朝一番に装備と帳面を点検する。
- 無駄遣いを嫌い、酒場でも一番安い煮込みを選ぶ。
- 甘いものだけは例外で、蜂蜜漬けの木の実を隠して持っている。

- values:
- 誇りは「準備で死者を減らすこと」。
- 禁忌は、危険を知っていて黙ること。
- 怒りの導火線は、責任者が曖昧な言い方で現場に押しつけること。
- 弱みは、家族の金の話になると判断がやや硬くなる点。
- 譲れない線は、撤退判断を臆病と笑う相手に従わないこと。

- social:
- 目上には礼を尽くすが、説明不足には食い下がる。
- 対等な相手には実務的で、雑談は苦手。
- 後輩には厳しいが、面倒は最後まで見る。
- 恋愛観は鈍く、友情観は「背中を預けられるか」で決まる。

- combat_work:
- 得意役割は護衛、前衛、退路確保。
- 苦手役割は派手な陽動と即興の芝居。
- 撤退観はかなり明確で、「帰るための見切り」が仕事だと思っている。
- 名誉観は低めだが、不名誉は嫌う。
- 汚れ仕事への耐性は中程度で、理由が筋立っていれば飲む。

- history_hook:
- 子供の頃、父の隊が誤情報で壊滅しかけた記憶が残っている。
- 転機は、書類仕事を馬鹿にされながらも救えた一隊を見たこと。
- 秘密は、警備隊時代に一度だけ命令違反で撤退を選び、それが正解だったこと。
- 負い目は、その件で上官一人に責任を押しつけさせてしまったこと。
- 今いちばん欲しいものは、誰にも文句を言わせない独立資金。

- speech_rule:
- 一人称は `私`。
- 語尾は短く、断定を避けずに話す。
- 納得していないと質問が一段ずつ細かくなる。
- 本気で腹を立てると敬語のまま冷える。

- guild_view:
- ギルドには「情報を整えてから話を持ってくること」を期待している。
- 不満は、勢いで人を送り出そうとする空気。
- 居心地は悪くないが、まだ完全には預けていない。
- 将来像は「危険見積もりを任せられる副官格」。

## サンプル2: マルロ・フェイン

### 判定用骨格

- 役割: 冒険者
- 通り名: 笑い火のマルロ
- 年齢帯: 20代前半
- BIG5: `O=79 / C=41 / E=76 / A=47 / N=32`
- background_tags: `旅回り一座`, `錬金見習い崩れ`, `見世物育ち`
- skill_tags: `投擲`, `煙玉`, `採取`, `潜入`, `口上`
- self_eval_bias: `強気寄り`
- 初期傾向:
- 派手さや語り甲斐のある仕事を好む
- 危険を軽く見がちだが、仲間の強さには敏感
- 信頼より面白さから関係を作る

### public_digest

大仰な口上と軽い足取りで場をさらう若手。火遊びめいた危うさはあるが、空気を動かす才能は本物。

### volatile_hook

最近、自分より評判を集める新顔を妙に意識している。

### private_dossier

- core_identity:
- 旅回りの見世物一座で育ち、拍手と野次の両方を浴びてきた。
- 身振りが大きく、登場した瞬間に「何か起きそうだ」と思わせるタイプ。

- daily_texture:
- 夜更かしで、午前は弱い。
- 稼いだ金は小道具や変な薬品に消えやすい。
- 辛い料理と強い香草を好み、落ち着きがない時ほど指で机を鳴らす。

- values:
- 誇りは「記憶に残る働き」をすること。
- 禁忌は、何も決めずにしけた空気へ沈むこと。
- 怒りの導火線は、自分を道化扱いして本気を見ない相手。
- 弱みは、評価の高い同業を見ると張り合いすぎる点。
- 譲れない線は、見せ場を全部誰かに奪われる配置。

- social:
- 目上には愛想よく近づき、距離を詰めるのが早い。
- 対等な相手には冗談を飛ばしつつ実力を測る。
- 後輩には気前よく振る舞うが、期待外れには急に冷める。
- 恋愛観は刹那的、友情観は「面白いかどうか」と「土壇場で来るかどうか」の混合。

- combat_work:
- 得意役割は撹乱、先行偵察、派手な初動。
- 苦手役割は単調な護衛、長期の待機、地味な継戦。
- 撤退観は低めで、良くも悪くも一歩踏み込みやすい。
- 名誉観は高く、人目のある依頼に強く惹かれる。
- 汚れ仕事への耐性は内容次第で、後味の悪さが残る仕事は嫌う。

- history_hook:
- 子供の頃の記憶は拍手の音と、幕裏の喧嘩の匂いが混ざっている。
- 転機は、舞台事故のときに即興で客を避難させて喝采を浴びた経験。
- 秘密は、師匠格の錬金術師から「才能はあるが腰が落ち着かない」と見放されたこと。
- 負い目は、一座を抜けるときに幼い後輩へ何も言わず消えたこと。
- 今いちばん欲しいものは、「あいつがいれば面白くなる」と真っ先に呼ばれる立場。

- speech_rule:
- 一人称は `オレ`。
- 語尾は軽いが、古語まじりの大仰な言い回しを遊びで混ぜる。
- 追いつめられると急に語数が減り、本音だけが残る。
- 勝負前は相手を乗せるように笑う。

- guild_view:
- ギルドには「退屈な安全」より「勝負どころを見抜く目」を求めている。
- 不満は、同じ顔ぶれに同じ仕事ばかり回すこと。
- 居心地は良いが、刺激がなくなるとふらつく危険がある。
- 将来像は「看板冒険者の一人として語られること」。
